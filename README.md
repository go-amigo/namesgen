### namesgen

Wrap github.com/docker/docker/pkg/namesgenerator into a main package
so we can call it from cli.


#### install

    $ go get github.com/go-amigo/namesgen
    $ $GOPATH/bin/namesgen

#### build locally

    $ go build .
    $ ./namesgen 2
    desperate_engelbart3
